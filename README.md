# Ng2Carouselamos

Ng2-carouselamos is a simple carousel/slider which just use css transitions to do the work

###Getting started
- Install node package:
```
npm i ng2-carouselamos --save
```

And then, in your app.module.ts :
```
...
@NgModule({
  ...
  imports: [
    ...
    Ng2CarouselamosModule
  ],
  ...
})
```
